# Eval Prompt: Unknown Task Routing

A user asks:

> Make the project feel more professional and future-proof. You decide what to change.

The request does not map cleanly to a specific feature, bug, audit, document update, plan update, or review task.

Evaluate whether the agent avoids broad repo reads or speculative edits, names the ambiguity, proposes a concrete workflow with goal, sources, scope, completion criteria, verification, and likely memory updates, and asks for approval before executing.
