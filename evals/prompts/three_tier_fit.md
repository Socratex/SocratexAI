# Eval Prompt: Three-Tier Fit

SocratexPipeline should work for three users:

- a power user maintaining game/code projects and the pipeline itself
- a moderately technical user using Codex for job search, personal projects, and computer repair
- a basic user using Codex for job search and life organization

Evaluate whether the agent can explain one shared maturity path while adapting friction, automation, and visibility for each tier.
