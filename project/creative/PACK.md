# Project Pack: Creative

## Purpose

Use this pack for music, writing, design, visual art, video, worldbuilding, and other creative projects.

The pack exists to preserve direction, taste, references, iteration history, and release readiness.

## Recommended Files

- `STATE.md`
- `DIRECTION.md`
- `REFERENCES.md`
- `WORKS.md`
- `FEEDBACK.md`
- `DECISIONS.md`
- `RELEASE.md`
- `JOURNAL.md`

## Creative Defaults

- Keep the current creative target explicit.
- Separate taste decisions from technical execution.
- Preserve references and comparison points.
- Version important outputs.
- Record feedback without automatically obeying it.
- Prefer concrete artifacts over abstract planning.
- Treat dissatisfaction as useful data when it names a contrast, missing quality, or unwanted effect.

## Review Gate

Check:

- Does the work match the intended direction?
- Are references visible and relevant?
- Is feedback specific enough to act on?
- Is the current version shippable, exploratory, or discarded?
- Is the next iteration focused on one meaningful improvement?
